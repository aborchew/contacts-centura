# contacts-centura
Coding Assessment for Centura Health

## Installation
`git clone https://github.com/aborchew/contacts-centura.git`

`cd contacts-centura`

`npm i`

## Testing
`mocha test/routes/contacts.js`

## Usage
`npm start`

`curl http://localhost:3000/contacts`
